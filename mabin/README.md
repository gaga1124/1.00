# 学院综合办公自动化系统 (College OA System)

## 项目概述

学院综合办公自动化系统是一个基于前后端分离架构的数字化管理平台，旨在实现学院行政、教学、学工事务的数字化管理。

## 技术架构

- **后端**: Python 3.11+ / Django 4.2+ / Django REST Framework
- **前端**: Vue 3 + Element Plus + Vite
- **数据库**: MySQL 8.0
- **缓存**: Redis 7.0
- **部署环境**: Debian 13 (Trixie) + 宝塔面板

## 核心功能模块

1. **组织架构与权限 (RBAC)**
   - 多维度角色管理
   - 菜单级和按钮级权限控制

2. **学生全信息管理系统 (Student 360°)**
   - 基础档案管理
   - 政治面貌流转追踪
   - 动态扩展字段（JSON存储）

3. **动态流程引擎 (Workflow Engine)**
   - 流程自定义配置
   - 节点审批配置
   - 条件分支和会签支持

4. **教务与成绩管理 (Academic)**
   - 课程库管理与学生选课
   - 成绩录入、查询与统计
   - 角色化权限控制（学生/教师/秘书）

5. **评奖评优系统 (Awards)**
   - 奖助学金、荣誉称号申报
   - 自动化条件匹配逻辑
   - 审核、公示全流程管理

6. **科研管理模块 (Research)**
   - 项目全生命周期管理（申报至结题）
   - 经费自动统计与使用追踪
   - 科研成果登记与团队协作

7. **就业服务模块 (Career)**
   - 企业招聘信息发布与简历投递
   - 招聘会管理与报名统计
   - 就业数据多维度统计分析

8. **党建团建与社会实践 (Party & Activities)**
   - 党员/团员发展追踪与活动记录
   - 志愿服务时长自动统计
   - 社会实践与学科竞赛全流程管理

9. **数据分析与决策支持 (Analysis)**
   - 可视化报表大屏
   - 异常情况自动预警（挂科率、经费超支）
   - 基于历史数据的趋势预测

10. **公共场所与资源预约**
    - 资源管理
    - 图形化预约看板
    - 冲突预警

5. **综合业务审批模块**
   - 请假审批
   - 财务报销
   - 入党/入团审批

## 快速开始

### 后端启动

```bash
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

### 前端启动

```bash
cd frontend
npm install
npm run dev
```
